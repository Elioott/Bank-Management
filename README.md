# 🏦 Site de Banque en ligne

Bienvenue sur notre une application web de banque en ligne moderne et sécurisée pour notre projet de fin d'année 2024/2025

## 🚀 Fonctionnalités

- 🔐 Authentification sécurisée 
- 📊 Suivi de budget visuel et intuitif
- 💸 Transferts entre comptes
- 🌐 Interface claire et moderne

## 🛠️ Technologies utilisées

- **Symfony** (framework principal)
- **PHP**
- **Twig** pour les templates HTML
- **CSS** pour le design
- **Docker Desktop**
- **PostgreSQL**
- **JavaScript**

## ❓Comment installer ce projet ❓

- Cloner le projet github dans votre espace avec la commande "git clone https://github.com/Elioott/Bank-Management.git"
- Installer les dépendances à la racine du projet "composer install"
- Modifier la ligne DATABASE_URL dans le .env
- Lancer avec Symfony "symfony serve ... "
